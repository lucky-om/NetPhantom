Intrusion Signature Index

NetPhantom incorporates active heuristic algorithms processing L2, L3, and L4 packets in real-time. Below are the registered attack signatures mapped by our detection engine.

HIGH SEVERITY PORT_SCAN
Heuristic: Target Range

Multi-Port Scan Discovery

Triggers when a single source host IP address attempts connections to more than 15 unique destination ports on a target system within a sliding time window.

Rule Signature: src_ip -> dst_port_count > 15
HIGH SEVERITY SYN_FLOOD
Heuristic: Flag Ratio

TCP SYN Flood Probe

Monitors the rate of TCP [SYN] handshake initiation packets that do not contain an [ACK] confirmation. Flagged when a source exceeds 50 SYN packets per second.

Rule Signature: tcp_flags == 'S' & pps > 50
HIGH SEVERITY ARP_SPOOF
Heuristic: Hardware Cache

ARP Cache Spoof / Poisoning

Detects changes in local ARP translations. Triggers when an incoming ARP reply claims ownership of an IP address that was previously registered to a different hardware MAC address.

Rule Signature: arp_table[ip] != mac_address
MEDIUM SEVERITY ICMP_FLOOD
Heuristic: Packet Rate

ICMP Flood (Ping Flood)

Identifies Denial-of-Service attempts targeting host processing capacity. Flagged when ICMP echo requests (pings) exceed 50 packets per second from one source.

Rule Signature: proto == 'ICMP' & pps > 50
MEDIUM SEVERITY DNS_FLOOD
Heuristic: Query Rate

DNS Query Flood

Flags potential query-amplification or endpoint exhaustion attacks. Triggered when DNS name resolution query attempts exceed 30 requests per second from a single IP.

Rule Signature: proto == 'DNS' & pps > 30
