Windows Release

Setup Installer

File: NetPhantom_Setup.exe

The official NetPhantom Windows setup installer executable (NetPhantom_Setup.exe). Upon execution, it automatically configures all NetPhantom packet engine modules, installs Desktop & Start Menu shortcuts with official logo icons, configures file associations, and sets up system environments.

Download Windows Setup (.exe) Download Setup (.zip) Size: ~67 MB
⚡ Windows 11 Smart App Control / Defender Notice:
If Windows Smart App Control blocks NetPhantom_Setup.exe:
  1. Right-click NetPhantom_Setup.exe → select Properties.
  2. At the bottom of the General tab, check Unblock → click Apply & OK.
  3. Or download NetPhantom_Setup.zip above (Zip archives bypass Smart App Control filters).
* Requires Npcap Driver installed on Windows host OS for promiscuous raw socket packet binding.
PyPI Package

Install via Python Pip

If you already have Python 3.9+ configured, install NetPhantom directly into your global virtual environment using pip:

pip install netphantom
netphantom --interface eth0
Dependencies automatically resolved: scapy, colorama, psutil.
Source Build

Clone & Build from GitHub

Inspect the underlying Python source code, modify parsing modules, or run raw scripts on Linux:

git clone https://github.com/lucky-om/NetPhantom.git
cd NetPhantom && pip install -r requirements.txt
python src/netphantom.py
