⚙ WHAT THE SETUP WIZARD DOES

The standalone setup installer automates the entire deployment process:

  • Extracts the pre-compiled desktop sniffer utility (NetPhantom.exe).
  • Appends system-wide Start Menu and Desktop shortcuts for quick access.
  • Automatically checks for Python and runs pip links to configure the global netphantom terminal command.
  • Ensures all config files, source modules, and setup assets are neatly stored inside a dedicated NetPhantom folder.
