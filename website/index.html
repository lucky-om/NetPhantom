Frequently Asked Questions

What is NetPhantom and how is it used?
NetPhantom is a premium local network sniffer and dissection analyzer designed for security auditing, penetration testing, and packet flow debugging. It mirrors network interface card data into an interactive GUI HUD console.
Why does packet capture require administrator access?
Sniffing sockets operates at a low hardware level (promiscuous mode) to capture incoming and outgoing raw network frames. Operating systems require root permissions (sudo on Linux, Administrator on Windows) to access raw network adapters.
What is the difference between NetPhantom and Wireshark?
While Wireshark is an industry-standard enterprise tool, NetPhantom provides a streamlined, dark-themed HUD layout with built-in stateful threat detection heuristics (SYN flood warnings, port scan discovery, ARP cache change logs) right out of the box.
Is NetPhantom safe to run? Does it send any data to external servers?
NetPhantom is 100% private and telemetry-free. All captured frames, dissected fields, and threat diagnostics are processed in local volatile RAM and are never cached or uploaded to remote endpoints.
